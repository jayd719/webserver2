Data Analysis | Workforce Data Visualization Project

Technologies: Python, Dash, Plotly, Pandas, Gunicorn

  • Built an interactive data dashboard to visualize employment trends from the 2021 Canadian Census using Python, Dash, and Plotly, enabling users to explore workforce data by region, occupation, education, and gender
  • Processed and cleaned large datasets (~100K+ records) by removing irrelevant columns, normalizing occupation and gender labels, and grouping education levels using Pandas, improving data quality and analysis efficiency.
  • Developed modular components including data models, charting functions, and front-end views to enhance scalability and maintainability of the application
  • Integrated interactive filters and dropdowns to allow users to segment data by province, occupation category, and education level, improving user engagement and insight discovery.
