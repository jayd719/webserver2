Exploring, Learning, and Applying: Developing My Skill Set

Development

Delving deeper into Django’s robust features. This included mastering Django’s ORM (Object-Relational Mapping), which simplifies database interactions, and learning to implement authentication, URL routing, and form handling. I also worked on creating RESTful APIs using Django REST framework to build scalable and maintainable web applications

Big Data

The world of Big Data is vast and constantly evolving. My journey began with understanding the fundamental concepts of Big Data, such as the 3 Vs: Volume, Velocity, and Variety. I explored various data processing frameworks and tools like Hadoop and Spark to grasp how massive datasets are managed and analyzed

CNCs

To deepen my knowledge, I studied G-code programming, which is essential for controlling CNC machines. I also learned about CAD (Computer-Aided Design) and CAM (Computer-Aided Manufacturing) software, which are used to design and translate those designs into machine-readable instructions. This phase involved hands-on practice with various CNC simulators and machines.
